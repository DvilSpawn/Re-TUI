# Fun Commands

Re:T-UI is not just a utility launcher.

It is also allowed to have taste.

## `hack`

The `hack` command is a terminal gimmick overlay that temporarily takes over the screen and prints a fake breach / system intrusion sequence.

It is not meant to do anything useful.

That is why it is good.

## Why Keep Things Like This?

Features like `hack` matter because they reinforce the identity of the launcher:

- not sterile
- not generic
- not pretending the terminal aesthetic is only decorative

## Possible Future Direction

This category is where future playful terminal features belong:

- animated ASCII boot sequences
- fake scans
- matrix-style overlays
- diagnostic rituals
- visual command effects

As long as they do not make the launcher unreliable, these features are part of the fun.
